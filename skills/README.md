# Skills 目录说明

`skills/` 目录存放供内部 AI 智能体调用的**领域知识技能文件**（`.md` 格式）。

这些文件会在每次任务运行时被加载，注入到所有智能体的 user message 头部（以技能列表形式），供智能体在分析中引用。

---

## 如何新增技能文件

1. **在 `skills/` 目录下创建 `.md` 文件**，文件名使用描述性名称，如 `semiconductor_analysis.md`

2. **文件格式要求**：

   推荐使用 YAML front matter 定义适用场景（用于技能列表展示）：

   ```markdown
   ---
   description: 半导体行业分析框架
   applicable_scenarios: 分析半导体/芯片板块个股时
   ---

   # 半导体行业分析框架

   ## 行业特征
   ...

   ## 关键指标
   ...
   ```

   或者使用 `## 适用场景` 段落：

   ```markdown
   # 消费电子分析框架

   ## 适用场景
   分析消费电子/智能手机产业链个股时

   ## 行业特征
   ...
   ```

3. **文件内容建议包含**：
   - 行业/场景的关键特征描述
   - 重要分析指标和阈值
   - 常见风险点
   - 历史规律和参考数据

4. **技能文件会自动被加载**，无需修改代码。系统启动后缓存技能列表，修改技能文件后重启服务生效。

---

## 技能文件的作用机制

```
每次 LLM 调用时：
  User message =
    （可用技能列表：
    [semiconductor_analysis.md] — 半导体行业分析框架
    [sector_rotation.md] — 板块轮动判断方法
    ---
    ）

    + 数据上下文（行情/指标/资金流等）
```

智能体看到技能列表后，可以在分析报告中引用相关技能内容，增强领域专业性。

## 示例技能文件结构

以下是一个技能文件的完整示例：

```markdown
---
description: A股量化选股因子参考手册
applicable_scenarios: 进行A股量化因子分析、多因子模型评估时
---

# A股量化选股因子参考手册

## 价值因子

### PE(TTM)
- 全市场中位数：约15-20倍（不同市场环境差异较大）
- 低估参考：PE < 10倍（需结合行业）
- 高估参考：PE > 50倍（成长股除外）

### PB(MRQ)
- 银行/地产低PB属正常：PB < 1为可接受
- 制造业合理区间：1-3倍
- 科技成长股：PB无固定参考

## 技术因子

### 均线金叉
- MA5上穿MA20：短期看涨信号
- MA20上穿MA60：中期趋势确认
- 三线多头排列（MA5>MA20>MA60）：强趋势

## 风险警示因子

### 高风险特征
- 融资余额占流通市值 > 15%：去杠杆风险
- 近3月有停牌记录：不确定性较高
- 北向资金连续净流出5日：外资撤离信号
```
