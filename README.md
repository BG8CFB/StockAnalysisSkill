# Stock Analysis Skill (股票多智能体量化分析)

本项目是一个基于多智能体（Multi-Agent）架构的股票量化分析流水线服务。通过 11 个 AI 智能体组成的 4 阶段流水线，对 A 股、港股、美股进行涵盖技术面、基本面、资金流和风控维度的深度分析，并输出结构化的投资研究报告。

本项目完全符合 **AI Agent Skills Open Specification** 标准，可直接作为 Claude Code、Cursor、Trae 等 AI 智能体的原生技能（Skill）使用。

## 🚀 快速开始 (人类用户必读)

如果你刚刚克隆或下载了本仓库，**无需手动配置**。你只需将本仓库目录拖入你喜欢的 AI IDE（如 Cursor, Windsurf, Trae）中，或在终端唤起 AI CLI（如 Claude Code），然后向 AI 发送以下这段“魔法提示词”：

> **✨ 复制以下内容发送给你的 AI：**
> 
> 请阅读项目根目录下的 `AI_SETUP.md` 文件，并严格按照其中的指示，帮我全自动完成这个项目的环境安装、依赖配置、端口冲突检查及服务启动。

AI 将会自动为你完成所有繁琐的配置工作（如果需要输入 API Key，AI 会主动询问你）。

## 📦 项目结构

- `SKILL.md`: 核心技能描述文件（符合 AI 智能体技能标准）。
- `AI_SETUP.md`: 专供 AI 阅读的自动化安装配置说明书。
- `src/`: 核心业务逻辑（数据获取、AI 智能体、API 路由）。
- `scripts/`: 快捷启动/停止脚本。
