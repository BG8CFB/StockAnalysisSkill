# 计算引擎规范

> 版本：v1.0
> 所属层：数据层（三层架构第一层）
> 职责范围：定义所有可计算指标的字段名、计算规则、评分规则。本文档是字段定义的唯一权威来源，工具层和智能体层均以本文档为准。

---

## 一、计算引擎总览

```
┌──────────────────────────────────────────────────────────────────┐
│                      计算引擎数据流                               │
│                                                                  │
│  输入（来自清洗后数据包）：                                         │
│    price_series（前复权价格序列）                                  │
│    daily_basic（PE/PB/换手率等）                                  │
│    capital_flow_raw（原始资金流）                                  │
│    margin_raw（原始融资融券）                                      │
│    anomalies（异常事件列表）                                       │
│                          ↓                                       │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                   计算引擎（纯代码确定性计算）             │    │
│  │                                                         │    │
│  │  技术指标计算 ─┐                                         │    │
│  │  量化因子计算 ─┤──→ 各指标字段 + 各维度0-100评分          │    │
│  │  资金流指标   ─┘                                         │    │
│  └─────────────────────────────────────────────────────────┘    │
│                          ↓                                       │
│  输出（交给工具层）：                                              │
│    技术指标全量（MACD/RSI/KDJ/布林带/均线/量价）                   │
│    量化因子（动量/价值/波动率/资金流评分）                          │
│    各维度综合评分（indicator_score等，均为0-100整数）               │
│                          ↓                                       │
│  数据工具读取计算结果，按智能体配置注入对应上下文                    │
└──────────────────────────────────────────────────────────────────┘
```

### 计算原则

- 所有可以用确定性公式计算的内容，均在本层完成，不交给智能体
- 智能体只接收计算结果，负责解读和判断，不重复计算
- 所有评分以 0-100 整数为单位，50 为中性基准

## 二、技术指标

所有技术指标基于**前复权收盘价**（`close_adj`）计算。

### 2.1 MACD

**参数**：快线12日 / 慢线26日 / 信号线9日

| 输出字段 | 含义 |
|--------|------|
| `macd_dif` | DIF线：快线EMA 减 慢线EMA |
| `macd_dea` | DEA线：DIF的9日指数移动平均（信号线） |
| `macd_bar` | 柱状图：2 × (DIF - DEA)，反映两线偏离程度 |
| `macd_signal` | 多空方向：DIF > DEA 为 `bullish`，否则为 `bearish` |
| `macd_cross` | 交叉事件：当日形成金叉为 `golden_cross`，死叉为 `death_cross`，无交叉为 `none` |
| `macd_cross_days` | 自上次金叉或死叉以来的连续天数（如"金叉第3日"） |

### 2.2 RSI

**参数**：14日

| 输出字段 | 含义 |
|--------|------|
| `rsi_14` | RSI数值，范围 0-100 |
| `rsi_signal` | 区间判断：`overbought`（>70）/ `strong`（55-70）/ `neutral`（45-55）/ `weak`（30-45）/ `oversold`（<30） |
| `rsi_divergence` | 背离信号：`bearish_divergence`（价格新高但RSI未新高）/ `bullish_divergence`（价格新低但RSI未新低）/ `none` |

### 2.3 KDJ

**参数**：RSV周期9 / K值平滑3 / D值平滑3

| 输出字段 | 含义 |
|--------|------|
| `kdj_k` | K值：RSV的加权移动平均 |
| `kdj_d` | D值：K值的加权移动平均 |
| `kdj_j` | J值：3K - 2D，超买超卖敏感度更高 |
| `kdj_signal` | 状态判断：`overbought`（K>80且D>80）/ `oversold`（K<20且D<20）/ `golden_cross`（K上穿D）/ `death_cross`（K下穿D）/ `neutral` |

### 2.4 布林带

**参数**：20日均值 / 2倍标准差

| 输出字段 | 含义 |
|--------|------|
| `bb_upper` | 上轨：中轨 + 2倍20日标准差 |
| `bb_middle` | 中轨：20日简单移动平均 |
| `bb_lower` | 下轨：中轨 - 2倍20日标准差 |
| `bb_bandwidth` | 带宽百分比：(上轨 - 下轨) / 中轨 × 100，反映波动收窄或扩张 |
| `bb_position` | %B指标：(收盘价 - 下轨) / (上轨 - 下轨)，0.5为中轨，>1为突破上轨，<0为跌破下轨 |
| `bb_signal` | 位置信号：`above_upper` / `below_lower` / `break_middle_up`（上穿中轨）/ `within_band` |

### 2.5 均线系统

**计算周期**：MA5、MA10、MA20、MA30、MA60、MA120、MA250

| 输出字段 | 含义 |
|--------|------|
| `ma_5` ... `ma_250` | 各周期简单移动平均值，历史数据不足时为 null |
| `ma_bullish_arrange` | 多头排列布尔值：MA5 > MA10 > MA20 > MA60 时为 true |
| `ma_bearish_arrange` | 空头排列布尔值：MA5 < MA10 < MA20 < MA60 时为 true |
| `ma_cross_5_10` | 5日与10日均线交叉：`golden`（5日上穿10日）/ `death`（5日下穿10日）/ `none` |
| `ma_cross_5_20` | 5日与20日均线交叉，同上 |
| `ma_cross_10_20` | 10日与20日均线交叉，同上 |
| `ma_cross_20_60` | 20日与60日均线交叉，同上 |
| `price_vs_ma20` | 价格偏离20日均线的百分比，(收盘 - MA20) / MA20 × 100 |
| `price_vs_ma60` | 价格偏离60日均线的百分比 |

### 2.6 成交量指标

| 输出字段 | 含义 |
|--------|------|
| `volume_ratio` | 量比：当日成交量 / 近5日平均成交量 |
| `amplitude` | 振幅：(最高价 - 最低价) / 前收盘价 × 100 |

## 三、量化因子

### 3.1 动量因子

**数据来源**：前复权收盘价序列

| 输出字段 | 含义 | 最小数据要求 |
|--------|------|-----------|
| `mom_1m` | 近1个月涨跌幅：最新收盘 / 20交易日前收盘 - 1，单位% | 20个交易日 |
| `mom_3m` | 近3个月涨跌幅：最新收盘 / 60交易日前收盘 - 1，单位% | 60个交易日 |
| `mom_6m` | 近6个月涨跌幅：最新收盘 / 120交易日前收盘 - 1，单位% | 120个交易日 |
| `mom_3m_skip1m` | 跳过近1个月的3月动量：20日前收盘 / 80日前收盘 - 1，单位%（规避短期反转偏差） | 80个交易日 |
| `mom_6m_skip1m` | 跳过近1个月的6月动量：20日前收盘 / 140日前收盘 - 1，单位% | 140个交易日 |
| `momentum_score` | 综合动量评分（0-100，50为基准） | — |

**动量评分规则**：

| 条件 | 加减分 |
|-----|-------|
| 1月动量 > 5% | +15 |
| 1月动量 < -5% | -15 |
| 3月动量 > 15% | +15 |
| 3月动量 < -10% | -15 |
| 6月动量 > 30% | +10 |
| 6月动量 < -20% | -10 |

最终分数限制在 0-100 范围内。历史数据不足时对应字段为 null，评分字段为 null。

### 3.2 价值因子

**数据来源**：每日基本面指标

| 输出字段 | 含义 | 说明 |
|--------|------|------|
| `pe_ttm` | 市盈率TTM | 为负或数据缺失时为 null |
| `pb_mrq` | 市净率MRQ | — |
| `ps_ttm` | 市销率TTM | — |
| `dividend_yield` | 股息率 | 单位% |
| `value_score` | 综合价值评分（0-100，50为基准，估值越低分数越高） | — |

**价值评分规则**：

| 条件 | 加减分 |
|-----|-------|
| PE < 10 | +20 |
| PE 在 10-20 | +10 |
| PE > 50 | -15 |
| PB < 1 | +15 |
| PB 在 1-2 | +5 |
| 股息率 > 3% | +10 |
| 股息率 > 5% | 再额外 +10 |

最终分数限制在 0-100 范围内。PE 为负（亏损企业）时，PE维度不计分。

### 3.3 波动率因子

**数据来源**：前复权收盘价序列日收益率

| 输出字段 | 含义 | 说明 |
|--------|------|------|
| `volatility_20d` | 20日年化波动率：20日日收益率标准差 × √252 × 100 | 单位% |
| `max_drawdown_20d` | 20日最大回撤：20日内相对最高点的最大跌幅 | 单位%，负值 |
| `volatility_score` | 波动率评分（0-100，50为基准，波动越低分数越高） | — |

**波动率评分规则**：

| 条件 | 加减分 |
|-----|-------|
| 年化波动率 < 20% | +25 |
| 年化波动率 20%-30% | +15 |
| 年化波动率 30%-40% | +5 |
| 年化波动率 > 60% | -20 |
| 年化波动率 > 80% | -35（含上一档的-20） |

最终分数限制在 0-100 范围内。

## 四、资金流向计算

**数据来源**：原始资金流向数据（仅A股可用）

**主力定义**：超大单（单笔成交 > 50万元）+ 大单（单笔成交 10-50万元）

| 输出字段 | 含义 | 说明 |
|--------|------|------|
| `main_net_today` | 当日主力净流入：超大单净流入 + 大单净流入 | 万元 |
| `main_net_3d` | 近3日主力净流入累计 | 万元 |
| `main_net_5d` | 近5日主力净流入累计 | 万元 |
| `main_net_10d` | 近10日主力净流入累计 | 万元 |
| `retail_net_today` | 当日散户净流入：小单买入 - 小单卖出 | 万元 |
| `main_ratio` | 主力资金占当日总成交额的比例 | % |
| `northbound_flow` | 北向资金净流入 | 万元，A股专属，港股/美股为 null |
| `capital_score` | 资金流向综合评分（0-100，50为基准） | — |
| `capital_signal` | 资金信号：`strong_inflow` / `inflow` / `neutral` / `outflow` / `strong_outflow` / `data_unavailable` | — |

**资金评分规则**：

| 条件 | 加减分 |
|-----|-------|
| 当日主力净流入 > 5000万 | +20 |
| 当日主力净流入 > 1000万 | +10 |
| 当日主力净流入 < -5000万 | -20 |
| 当日主力净流入 < -1000万 | -10 |
| 5日累计净流入 > 1亿 | +15 |
| 5日累计净流入 < -1亿 | -15 |
| 北向资金净流入 > 1亿 | +10 |
| 北向资金净流出 > 1亿 | -10 |

（加分项互斥取最高档，不叠加）最终分数限制在 0-100 范围内。

**信号映射规则**：

| 评分区间 | 信号 |
|--------|------|
| ≥ 80 | `strong_inflow` |
| 65-79 | `inflow` |
| 36-64 | `neutral` |
| 21-35 | `outflow` |
| ≤ 20 | `strong_outflow` |

**数据不可用处理**：港股/美股无资金流向数据，所有资金字段为 null，`capital_score` 为 null，`capital_signal` 为 `data_unavailable`。

## 五、计算流程与顺序

计算引擎按以下顺序串行执行，前序结果作为后续输入：

| 步骤 | 内容 | 依赖 |
|-----|------|------|
| 1 | 技术指标计算：MACD → RSI → KDJ → 布林带 → 均线系统 → 成交量指标 | 前复权价格序列 |
| 2 | 量化因子计算：动量 / 价值 / 波动率（可并行） | 前复权价格序列 + 每日基本面 |
| 3 | 资金流向计算：各期净流入 + 评分 + 信号 | 原始资金流向数据 |
| 4 | 数据质量检查：统计缺失字段，输出质量等级 | 所有上述结果 |
| 5 | 格式化输出：整合为标准数据包，注入 LangGraph state | 所有上述结果 |

## 六、与智能体的边界说明

| 工作内容 | 负责方 | 说明 |
|---------|-------|------|
| 指标数值计算 | 计算引擎 | 确定性公式，代码实现 |
| 评分计算 | 计算引擎 | 规则确定，代码实现 |
| 信号分类（如 golden_cross） | 计算引擎 | 规则确定，代码实现 |
| 指标解读与综合判断 | 智能体 | 需要经验和逻辑推理 |
| 多维度矛盾裁决 | 智能体 | 需要权衡和综合分析 |
| 最终方向判断 | 智能体 | 需要综合所有维度 |
