## 二、VaR 计算

### 2.1 什么是 VaR

VaR（Value at Risk，风险价值）：在特定置信水平下，持有某仓位的特定天数内，可能发生的最大损失金额。

例如：VaR(95%, 10日) = 8000元，表示未来10个交易日内，有95%的概率亏损不超过8000元。

### 2.2 计算方法：历史模拟法

**选用原因**：股票收益率分布存在厚尾、偏态，历史模拟法比正态分布假设更接近真实风险。

**计算步骤**：

| 步骤 | 内容 |
|-----|------|
| 1 | 取历史前复权收盘价，计算每日收益率 |
| 2 | 取历史收益率分布的第5百分位数（即最差的5%的日收益率） |
| 3 | 持有期缩放：持有期VaR收益率 = 日VaR收益率 × √持有期天数（假设收益率独立） |
| 4 | 转换为金额：VaR金额 = 持仓市值 × |持有期VaR收益率| |

**核心公式**：`VaR = 持仓市值 × |收益率第5百分位数| × √持有期`

### 2.3 输入参数

| 参数 | 说明 |
|-----|------|
| 股票代码 | 分析标的 |
| 持仓市值 | 该笔交易的计划持仓金额（元） |
| 持有期 | 计划持有天数（默认10个交易日） |
| 历史价格序列 | 前复权收盘价（至少120个交易日） |
| 置信水平 | 默认95% |

### 2.4 输出字段

| 输出字段 | 含义 |
|--------|------|
| `var_daily_pct` | 日VaR百分比（绝对值），如 1.8% 表示单日可能最大损失1.8% |
| `var_holding_pct` | 持有期VaR百分比（绝对值） |
| `var_amount` | 持有期VaR金额（元） |
| `confidence_level` | 置信水平（默认0.95） |
| `holding_days` | 持有期天数 |
| `exceeds_threshold` | 持有期VaR百分比是否超过系统阈值（10%）布尔值 |

### 2.5 风控阈值

| 阈值项 | 数值 | 超过时的处理 |
|-------|-----|------------|
| VaR(95%, 10日) 上限 | 持仓市值的 10% | 超过则建议按比例降仓，使VaR降至 8% 以内 |

## 三、A股特有风险检查

**适用范围**：仅 A股（股票代码后缀为 .SZ 或 .SH）。港股、美股不执行本检查。

### 3.1 T+1 制度风险

**风险来源**：今日买入无法今日卖出，若判断有误，次日才能止损，隔夜敞口不可避免。

| 检查条件 | 风险评分加分 | 说明 |
|---------|-----------|------|
| 20日年化波动率 > 30% | +20 | 历史波动率高，隔夜跳空风险大 |
| 近60日最大隔夜低开幅度绝对值 > 5% | +15 | 有历史大幅跳空低开记录 |
| 3日内有业绩公告 | +25 | 业绩不确定性叠加隔夜风险，风险极高 |

**输出字段**：

| 字段 | 含义 |
|-----|------|
| `t1_risk_score` | T+1风险评分（0-100） |
| `t1_risk_level` | low / medium / high / extreme |
| `max_overnight_gap` | 近60日最大隔夜低开幅度，负值，单位% |
| `upcoming_events` | 未来3日内重大事件列表，如"业绩公告" |

### 3.2 涨跌停流动性风险

**风险来源**：涨停时追入，若次日开板可能大幅低开；跌停时无法出场，被迫持有。

| 检查条件 | 风险评分加分 | 说明 |
|---------|-----------|------|
| 当日涨停且封单金额 < 1000万元 | +20 | 封单薄弱，开板风险高 |
| 市场跌停家数 > 100 | +30 | 系统性下跌风险，个股流动性可能恶化 |
| 市场涨停家数 > 200 | 记录警告，不加分 | 情绪极度贪婪，潜在反转风险 |

**输出字段**：

| 字段 | 含义 |
|-----|------|
| `limit_up_risk_score` | 涨跌停流动性风险评分（0-100） |
| `is_limit_up` | 当日是否涨停 |
| `is_limit_down` | 当日是否跌停 |
| `limit_up_amount` | 涨停封单金额（万元），无涨停时为0 |

### 3.3 停牌风险

**风险来源**：持仓期间突发停牌，资金锁定，无法操作。

| 检查条件 | 处理方式 | 风险评分加分 |
|---------|---------|-----------|
| 当日已停牌 | 立即返回"禁止交易"，跳过后续所有检查 | 不适用 |
| 近90日内有停牌记录 | 记录风险提示 | +10 |
| 处于业绩窗口期（季末后45天内） | 记录风险提示 | +15 |

**输出字段**：

| 字段 | 含义 |
|-----|------|
| `suspend_risk_score` | 停牌风险评分（0-100，停牌时不适用） |
| `is_suspended_today` | 当日是否停牌 |
| `in_earnings_window` | 是否处于业绩窗口期 |
| `recent_suspend_count` | 近90日停牌次数 |

### 3.4 融资融券风险

**风险来源**：高融资余额意味着杠杆资金堆积，一旦触发去杠杆，将出现踩踏式下跌。

**注意**：融资融券数据延迟 T+1，当日分析使用前一个交易日数据。

| 检查条件 | 风险评分加分 | 说明 |
|---------|-----------|------|
| 融资占流通市值比 > 15% | +30 | 杠杆资金集中，踩踏风险高 |
| 融资余额5日降幅 > 20% | +25 | 去杠杆正在进行中 |
| 融资占比 > 10% 且 5日降幅 > 10%（两条件同时满足） | 额外 +20 | 高融资 + 快速去杠杆，风险叠加 |

**输出字段**：

| 字段 | 含义 |
|-----|------|
| `margin_risk_score` | 融资风险评分（0-100） |
| `margin_ratio` | 融资余额占流通市值比（%） |
| `margin_change_5d` | 5日融资余额变化率（%，负值为减少） |
| `data_available` | 该股是否有融资资格（无融资资格标的为 false，不参与评分） |

### 3.5 综合风险汇总

**加权规则**：

| 风险维度 | 权重 | 权重依据 |
|---------|-----|---------|
| 融资融券风险 | 40% | 杠杆踩踏是最直接的系统性下跌驱动 |
| T+1制度风险 | 25% | 隔夜持仓核心风险，无法规避 |
| 涨跌停流动性风险 | 20% | 流动性陷阱风险 |
| 停牌风险 | 15% | 相对可预判，可通过事前筛查部分规避 |

**交易建议映射**：

| 综合评分 | 交易建议 | 含义 |
|---------|---------|------|
| ≥ 70 | 否决交易 | 风险过高，建议不执行本次交易 |
| 50-69 | 降仓执行 | 将仓位降至原计划的 70%，谨慎执行 |
| 30-49 | 谨慎可接受 | 可正常执行，但需在报告中充分披露风险 |
| < 30 | 风险可控 | 正常执行 |

**当日停牌时**：整个检查立即返回"禁止交易"，Stage 3 所有 AI 风控智能体不再启动，编排层直接输出停牌提示。
