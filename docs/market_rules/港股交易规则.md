# 【市场规则】港股交易规则

> 本文件由编排层在运行时注入至处理港股（代码后缀 .HK）的智能体系统提示词。
> 智能体必须严格遵循本文件中的市场规则，不得使用训练知识中与本文件冲突的内容。

---

## 一、基础交易制度

### 1.1 结算制度

| 制度项 | 规则 | 对分析的影响 |
|-------|------|------------|
| **T+2 结算** | 买入2个交易日后资金和股票才正式交割 | 理论上可当日卖出（T+0交易），但资金清算需2天 |
| **T+0 交易** | 同一账户同一股票，当日可买入后立即卖出（不同于A股T+1） | 可当日止损，消除A股的"次日被迫持有"风险 |
| **无涨跌停** | 港股无涨跌停限制，理论上可在一天内跌至零 | 风险敞口无上限；止损委托应立即触发，无A股封板困境 |

**与A股的关键区别**：港股 T+0 交易意味着分析报告无需特别考量"今日买入无法今日出场"的情形；但无涨跌停限制意味着极端行情下单日亏损可能超过 20%，VaR 评估需更保守。

### 1.2 无涨跌停机制的影响

**正面**：
- 价格可以更快反映信息，减少信息滞后
- 止损可以即时执行，不存在封板困境
- 做空可以充分发挥价格发现功能

**负面**：
- 极端利空下，单日跌幅可能超过 50%（如财务造假曝光）
- 庄股风险：操控者可快速拉升后快速出货，散户无保护机制
- 流动性枯竭时，买卖差价（Spread）可能极大，止损成交价大幅偏离

### 1.3 港股做空机制

港股存在较完善的做空机制，这与A股有根本区别：

| 做空方式 | 说明 | 对个股分析的影响 |
|---------|------|---------------|
| **卖空（Short Selling）** | 机构可直接借券卖出，散户需通过券商 | 做空压力真实存在，不可忽视 |
| **股票期权** | 港股有成熟期权市场，包含认购/认沽 | 期权持仓可反映机构方向预判 |
| **沽空报告** | 知名做空机构（Muddy Waters等）公开沽空报告 | 一旦发布，股价可在数小时内暴跌30%+ |

**分析含义**：港股研究必须关注沽空比例；高沽空比例个股风险评分应显著提高。

### 1.4 手数限制（Lot Size）

港股每只股票最小交易单位（手数）不同，可能影响实际仓位执行：
- 蓝筹股（如腾讯）：通常 100 股/手
- 中小型股：可能 1000-10000 股/手
- **对策**：实际仓位可能因手数限制而略有偏差，交易计划中应说明

## 二、港股特有风险清单

### 2.1 汇率风险（港元/人民币）

港股以港元计价，内地投资者持有港股面临汇率风险：

| 汇率情形 | 对持仓影响（以人民币计） |
|---------|----------------------|
| 港元升值 | 人民币回报高于港元回报，正面 |
| 港元贬值 | 人民币回报低于港元回报，负面 |
| 港元与美元挂钩（联系汇率制度） | 美元强弱间接影响港元，但波动范围有限（7.75-7.85区间） |

**分析要求**：持有期超过1个月的中线/长线交易，需在风险提示中说明汇率风险。短线（< 5交易日）：汇率影响通常可忽略。

### 2.2 流动性分层风险

港股市场流动性分化极为严重：

| 市值分层 | 流动性风险 |
|---------|----------|
| 超大盘（> 500亿港元市值） | 低 |
| 大盘（100-500亿） | 中低 |
| 中盘（10-100亿） | 中等 |
| 小盘（< 10亿） | 高，持仓需谨慎 |
| 仙股（< 1亿港元市值） | 极高，不建议参与 |

**建议**：流动性低的股票，买入容易、卖出困难；止损计划中需考虑大额卖单的市场冲击。

### 2.3 停盘风险

港股无A股的行政停牌机制，但以下情形仍可能导致交易中断：
- 公司申请短暂停牌等待公告发布（通常数小时至1天）
- 极端波动触发交易所主动暂停（罕见）
- 公司除牌（最终结果是股票清零，投资者损失全部本金）

**除牌风险**：港股对长期亏损、不达标公司会除牌，与A股的"壳价值"保护完全不同。持有濒临除牌标准公司的风险等级应标注为"极高"。

### 2.4 庄股风险

港股中小型股庄股操控更为普遍，特征包括：
- 股价长期横盘后突然拉升，无基本面支撑
- 换手率异常，但成交多为对敲
- 沽空比例极低（庄家持有大量股票，难以借券）
- 突然暴跌并无法恢复

**识别标志**：股东集中度极高（前10大股东持股 > 80%）+ 无实质业务的"壳公司" → 直接回避。

## 三、港股市场特殊事件日历

### 3.1 影响港股的主要宏观事件

| 事件类型 | 影响机制 | 风险等级 |
|---------|---------|---------|
| **美联储议息会议** | 港元联系汇率制，港股资金成本直接受美联储影响 | 高 |
| **中国内地重大政策** | 大量港股业务在内地，政策影响直接 | 高 |
| **人民币汇率大幅波动** | 影响内地赴港资金量，也影响港股估值 | 中高 |
| **地缘政治（中美关系）** | 港股估值折价扩大/缩小 | 中高 |
| **香港本地政治/社会事件** | 影响市场情绪和外资配置意愿 | 中 |

### 3.2 港股财报披露规律

- 中期业绩（半年报）：通常 8月发布
- 年报：通常次年3-4月发布
- **与A股区别**：港股无强制季报，信息披露频率低于A股，导致业绩期波动更集中

### 3.3 港股交易时间（香港时间，UTC+8）

| 时段 | 时间 | 说明 |
|-----|------|------|
| 竞价时段 | 09:00-09:20 | 开盘前竞价 |
| 持续交易（上午） | 09:30-12:00 | 正常交易 |
| 持续交易（下午） | 13:00-16:00 | 正常交易 |
| 收市竞价 | 16:00-16:10 | 收盘竞价，可撤单 |
| 延长收市竞价 | 16:10-16:12 | 不可撤单 |

**注意**：港股午休时间为 12:00-13:00，A股午休为 11:30-13:00，两市重叠时间需注意。
