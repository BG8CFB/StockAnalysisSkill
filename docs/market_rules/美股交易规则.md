# 【市场规则】美股交易规则

> 智能体必须严格遵循本文件中的市场规则，不得使用训练知识中与本文件冲突的内容。

## 一、基础交易制度

### 1.1 结算制度

| 制度项 | 规则 | 对分析的影响 |
|-------|------|------------|
| **T+1 结算** | 2024年5月起美股正式迁移至T+1结算 | 资金结算周期缩短，流动性改善 |
| **T+0 交易** | 同一账户可当日买卖同一股票，但受 PDT 规则限制 | 理论上可当日止损，实际受 PDT 限制 |
| **无涨跌停** | 美股无涨跌停限制（但有熔断机制） | 极端情形下单日跌幅无上限，VaR 保守评估 |

### 1.2 PDT 规则（Pattern Day Trader，频繁日内交易者规则）

这是美股最重要的制度约束之一，与A股/港股有根本区别：

| 规则项 | 内容 |
|-------|------|
| **触发条件** | 保证金账户中，5个交易日内日内交易（当日开仓当日平仓）达到4次或以上 |
| **触发后限制** | 账户净值须维持 25,000 美元以上，否则只能进行平仓操作 |
| **适用账户** | 保证金账户（Margin Account）；现金账户不受 PDT 限制，但资金需等待结算 |
| **应对方式** | 账户净值 > 25,000 美元可无限日内交易；或使用现金账户；或持仓过夜 |

**分析含义**：账户净值较小的投资者，交易计划应优先考虑持仓隔夜，减少日内交易次数；止损时需考虑 PDT 限制是否允许当日再入场。

### 1.3 熔断机制

| 触发条件 | 影响 |
|---------|------|
| 标普500单日下跌7%（Level 1） | 全市场暂停交易15分钟 |
| 标普500单日下跌13%（Level 2） | 全市场暂停交易15分钟 |
| 标普500单日下跌20%（Level 3） | 当日剩余时间停止交易 |

**个股层面**：个股也有独立的熔断机制（LULD），防止个股在5分钟内过度波动，但不等同于涨跌停。

### 1.4 盘前盘后交易（Extended Hours）

| 时段 | 时间（美东时间） | 特点 |
|-----|--------------|------|
| 盘前交易 | 04:00-09:30 | 流动性低，价差大，重要新闻在此时段反应 |
| 正常交易 | 09:30-16:00 | 主要交易时段 |
| 盘后交易 | 16:00-20:00 | 流动性低，财报通常在盘后发布，波动剧烈 |

**关键点**：美股财报通常在盘后（After Market）或盘前（Pre Market）发布，导致次日开盘大幅跳空。财报日前后的隔夜风险极高。

## 二、美股特有风险清单

### 2.1 财报日历风险（Earnings Calendar Risk）

美股最重要的特有风险之一。

| 阶段 | 风险特征 |
|-----|---------|
| **财报发布前** | 期权隐含波动率大幅上升，预期拉满 |
| **财报发布后** | 若不及预期，股价可单日下跌10-30%；若超预期可上涨10-20% |
| **财报季** | 每季末后6周为财报密集期，市场整体波动上升 |

**财报季时间表**：
- Q1 财报：通常4月中旬至5月初
- Q2 财报：通常7月中旬至8月初
- Q3 财报：通常10月中旬至11月初
- Q4（全年）财报：通常次年1月至2月

**分析规则**：
- 财报发布日及前1日：持仓风险等级强制升为"高"，建议降低仓位
- **禁止在财报日前建立大仓位**，除非报告明确说明财报预期且做了情景分析

### 2.2 汇率风险（美元计价）

| 场景 | 人民币计算影响 |
|-----|------------|
| 美元升值 | 美股人民币回报优于美元回报，正面 |
| 美元贬值 | 美股人民币回报低于美元回报，负面 |
| 美联储加息 | 通常推升美元，同时对股市有压制效应 |
| 美联储降息 | 通常推升股市（流动性改善），美元或走弱 |

**分析要求**：中线（1个月以上）持仓必须在风险提示中说明汇率敞口。

### 2.3 利率风险

美股与无风险利率（10年期美债收益率）高度相关：
- **利率上升**：折现率上升，成长股（高PE）估值承压最大；价值股相对抗跌
- **利率下降**：成长股受益，科技股往往大涨
- **信用利差扩大**：经济衰退预期，高收益债压力传导至成长股

**分析要点**：美股分析不能仅看技术面，需结合美联储政策周期和国债收益率走势。

### 2.4 做空机制与轧空风险

美股做空机制完善，且机构参与度高：

| 做空形式 | 说明 |
|---------|------|
| **借券卖空** | 机构借券卖空，是主流机制 |
| **Put 期权** | 购买认沽期权，下跌受益 |
| **做空报告** | 专业做空机构发布做空报告，可引发股价暴跌 |
| **轧空（Short Squeeze）** | 高沽空比例个股若股价上涨，空头被迫买入平仓，形成正反馈暴涨 |

## 三、美股市场特殊事件日历

### 3.1 高影响宏观事件

| 事件 | 发布频率 | 对美股影响 |
|-----|---------|----------|
| **美联储议息会议（FOMC）** | 每年8次 | 高；利率决定和声明措辞驱动大盘 |
| **非农就业报告（NFP）** | 每月第一个周五 | 高；就业数据影响美联储政策预期 |
| **CPI/PCE 通胀数据** | 每月 | 高；影响美联储加息/降息路径 |
| **GDP 数据** | 每季度 | 中高 |
| **地缘政治事件** | 不定期 | 高时效性，无法预测 |

### 3.2 美股交易日历特殊说明

| 场景 | 影响 |
|-----|------|
| **美国联邦假日** | 全市场休市（注意国内节假日与美国节假日不同步） |
| **财报密集周** | 单周内大量大盘股发布财报，市场整体波动上升 |
| **季末重仓调整（Window Dressing）** | 基金季末前买入强势股、卖出弱势股 |
| **期权到期日（OpEx）** | 每月第三个周五，大量期权到期，可能影响相关股票价格 |
| **三重巫节（Triple Witching）** | 每季度第三个周五，股票期货/期权同时到期，波动极大 |

### 3.3 美股 vs A股 关键差异速查

| 维度 | A股 | 美股 |
|-----|-----|------|
| T+1/T+0 | T+1，当日不可卖出 | T+0，可当日止损（受PDT限制） |
| 涨跌停 | 有（±10%/±20%） | 无（有熔断机制） |
| 做空难度 | 极高（融券稀缺） | 较易（机构可直接借券） |
| 财报频率 | 季报+半年报+年报 | 季报+年报（更规范） |
| 主要驱动力 | 政策+情绪 | 业绩+美联储政策 |
| 汇率风险 | 人民币计价，无汇率风险 | 美元计价，存在汇率风险 |
